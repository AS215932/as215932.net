about

AS215932 is a solo project to build and operate a full-stack IPv6-first ISP from scratch: multi-homed BGP, RPKI, IRR, NAT64/DNS64 for legacy reachability. Configuration and runbooks are public on GitHub.

If you want to peer, see the peering page. For operational issues, email noc@as215932.net.
